This project showcases a comprehensive **web-based internal system** developed to support Technical Operation Support activities in a broadcasting environment. The system centralizes multiple operational tools into one platform, enabling efficient monitoring, reporting, and asset management.

## Key Features

- **Log Report & Checklist System**  
  Allows operators to input daily maintenance logs, shift reports, and checklist data to ensure all operational activities are recorded and traceable.

  <img src='/images/601.png' alt='Log Report dan Checklist'>

- **Inventory Management System**  
  Displays a structured list of assets including barcode, serial number, location, status, and user. Helps track equipment usage and condition in real-time.

  <img src='/images/602.png' alt='List Inventory'>

- **Technical Knowledge Base (Troubleshooting System)**  
  Provides documentation such as troubleshooting guides, operational procedures, and technical solutions for quick issue resolution by the team.

  <img src='/images/603.png' alt='Knowledge Base System'>

- **IP Mapping & Network Configuration Data**  
  Stores important network configuration details such as IP addresses, user credentials, VLAN, and port mapping for operational devices.

  <img src='/images/604.png' alt='Detail Mapping'>

- **PC Information Monitoring**  
  Displays detailed information about PCs used in operations, including division, IP address, and notes related to network or system usage.

  <img src='/images/605.png' alt='Informasi PC'>

## Technologies Used

- **Backend**: PHP (server-side processing), Python (data processing if needed)
- **Frontend**: HTML, CSS, JavaScript
- **Database**: MySQL (data storage for logs, inventory, and system info)
- **Design**: Responsive UI for operational environments
- **System Type**: Internal Web-Based Dashboard

## Outcome

This system successfully:

- Centralizes operational data into a single platform  
- Improves efficiency in logging and reporting technical activities  
- Reduces human error in asset and system tracking  
- Speeds up troubleshooting with built-in documentation  
- Enhances coordination between technical teams  

The platform is scalable and can be further developed into a fully integrated operational monitoring system for broadcasting or enterprise environments.
